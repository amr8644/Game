
# Snake Game

A snake game using C and SDL.
## Run Locally

Clone the project

```bash
 https://github.com/amr8644/Snake-game
```

Go to the project directory

```bash
  cd Snake-Game
```


Build project

```bash
  gcc -o main main.c entity.c render.c snake.c  -I /usr/local/include/SDL2 -L /usr/local/lib -lSDL2 -lSDL2_image

```

Run project

```bash
./main
```
