
# Snake Game

A snake game using C and SDL.
## Run Locally (Only works in Linux)

Clone the project

```bash
 https://github.com/amr8644/Snake-game
```

Go to the project directory

```bash
  cd Snake-Game
```


Build project

```bash
  make

```

Run project

```bash
./main
```
